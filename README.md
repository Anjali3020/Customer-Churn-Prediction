# Customer Churn Prediction

## Overview

This project predicts whether a customer is likely to leave a company (customer churn) using Machine Learning in Python. It demonstrates data preprocessing, model training, prediction, and evaluation using a sample customer dataset.

---

## Project Structure

```
Customer-Churn-Prediction/
│
├── Customer_Churn_Prediction.ipynb
├── customer_churn_10_features_100_rows.csv
└── requirements.txt
```

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas

- Seaborn
- Scikit-learn

## Features

- Load customer dataset
- Data preprocessing
- Train Machine Learning model
- Predict customer churn

The dataset contains customer information with multiple features used to predict customer churn.
